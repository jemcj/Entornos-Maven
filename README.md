INSTRUCCIONES DE EJECUCION

Requisitos

  - JDK 17 (o superior)
  - Maven 3.8 (o superior)

Ejecutar
Si se abre el proyecto desde Visual Stidio Code, simplemente con pulsar el boton de play en la parte superior derecha, estando dentro del archivo App.java es suficiente
Si no se abre desde ahi, se puede abrir con el comando --com.entornos.App--

DESCRIPCION

La aplicacion gestiona pedidos para una ampresa que vende libros, revistas, etc... de manera fisica y virtual. 
El sistema premite:
   - Registrar clientes
   - Gestionar productos (fisicos o digitales)
   - Crear pedidos y asociarlos a un cliente
   - Añadir productos a un cliente
   - Calcular el precio final de cada producto
   - Obtener una "factura"
